## 3 步核验到账（新手照做就行）

1. **粘贴地址**（TRON 地址一般是 `T` 开头）  
2. **看余额**：USDT/TRX 是否变化  
3. **看交易记录**：是否有对应转入（重点核对 **方向 / 金额 / 时间 / 对手地址**）

如果你只做到账核验，99% 的问题都能用这三步解决。

## 常见问题（FAQ）

### 1）对方说“已转账成功”，但我没收到？
先别吵，按三步核验：  
- **余额没变** + **交易记录没那笔** → 大概率没到/转错地址/链选错  
- **余额变了** + **交易记录有对应转入** → 到账（以链上为准）

### 2）为什么没有提示/不到账？
常见原因：链上确认延迟、地址填错、网络（链）选错、或截图话术。  
**不要猜，直接查余额和交易记录。**

### 3）必须要交易哈希（TxHash）才能查吗？
不一定。  
**从地址入手**更直观：你能看到一个时间段内所有收支流水，确认是否有对应转入。
