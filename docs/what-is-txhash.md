# 交易哈希（TxHash）是什么？没有哈希也能核验

TxHash（交易哈希）可以理解为每笔转账的“唯一单号”。有哈希当然更方便，但新手核验到账并不一定需要它。

## 没有哈希怎么办？
从地址入手更直观：
- 看余额是否变化
- 看交易记录是否有对应转入（方向/金额/时间）

## 什么时候一定要哈希？
- 对方坚持说“转了”，但你地址上查不到对应记录  
- 需要定位一笔很久以前的转账  
- 需要做纠纷举证时（哈希是强证据）

## 一句话总结
> 哈希是单号，地址是账户。核验到账：先查地址的余额和明细。
